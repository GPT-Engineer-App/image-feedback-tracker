# image-feedback-tracker

Create a simple app that
1. Let's the user upload images
2. Annotate them as good, okay, bad, or terrible
3. Keep track of the results in a database

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/image-feedback-tracker.git
cd image-feedback-tracker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
